PRUEBA DE CURSO GIT CON PILDORAS INFORMÁTICAS

esto es una prueba cuando uno detecta un error tipográfico, justo después de haber hecho commit y quieres corregirlo sin necesidad de hacer otro commit. Por ejemplo escribrir CANPO con N. esta correción despues de hacer commit, se hace arreglando el error, guardando cambios y yendo a la terminal de git escribir el comando: git commit ---amend --no-edit
